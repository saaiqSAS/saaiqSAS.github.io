A Complete Remake of SAS-STE-Wizard Coming Soon With New Features !!

A full remake of the SAS-STE-Wizard embedded with the first ever SAS-STE API will be released soon. This new version of SAS-STE-Wizard will support multiple file inputs and directory inputs. This would allow you to encrypt/decrypt all the text based files within a directory by giving the path of the directory itself. In addition to this, the upcoming version would allow you to change StaticKeys within the wizard without any external processes.

The new version will also provide a better CLI wizard flow, reducing the number of stops in the wizard to take user's choices, resulting in the reduction of the time taken for the use of SAS-STE-Wizard. Last but not Least, the new version of the SAS-STE-Wizard will be using the newly improved SAS-STE v4 algorithm with better security and a more wider range of unique keys. Followed by this major update would be the update of all other SAS-STE v3 releases inorder to support SAS-STE v4. Worth noting that SAS-STE v4 will be backwards compatible with SAS-STE v3.
