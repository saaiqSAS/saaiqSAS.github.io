SAS-STE-Web Now Supports SAS-STE v4

SAS-STE-Web is now updated to support the newly released SAS-STE v4 algorithm which provides better security than the older SAS-STE v3 algorithm. The v4 algorithm will be backwards compatible with the v3 algorithm.

A Complete Remake of SAS-STE-Wizard (v4.4.6) Now Released With Stunning Features!!

A full remake of the SAS-STE-Wizard (v4.4.6) embedded with the first ever SAS-STE API is now released. This new version of SAS-STE-Wizard will support multiple file inputs and directory inputs. This would allow you to encrypt/decrypt all the text based files within a directory by giving the path of the directory itself. In addition to this, v4.4.6 would allow you to change StaticKeys within the wizard without any external processes.

SAS-STE-Wizard v4.4.6 will also provide a better CLI wizard flow, reducing the number of stops in the wizard to take user's choices, resulting in the reduction of the time taken for the use of SAS-STE-Wizard. Last but not Least, SAS-STE-Wizard v4.4.6 will be using the newly improved SAS-STE v4 algorithm with better security and a more wider range of unique keys.

Followed by this major update will be the update of all other SAS-STE v3 releases inorder to support SAS-STE v4. Also Worth noting that the SAS-STE v4 will be backwards compatible with SAS-STE v3. Moreover, a brand new, much more detailed documentation of SAS-STE is now being written and will soon be available
