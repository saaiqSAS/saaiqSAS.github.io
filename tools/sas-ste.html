SAS-STE : Secure Text Encryption is a text based encryption algorithm designed and made by Saaiq Abdulla Saeed based on a unique concept. SAS-STE supports the encryption of pure plain text and any text based files. In addition to supporting keyed and no key encryptions, SAS-STE allows you to re use previously generated keys. SAS-STE algorithm uses two types keys to both encrypt and decrypt text based data. Static keys, which are stored in the program and Public keys, which are used in general. To find more information about SAS-STE and how it works, please refer to the documentation.
