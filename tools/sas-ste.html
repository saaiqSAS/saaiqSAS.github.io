SAS-STE

SAS-STE or SAS-Secure Text Encryption is a text based encryption algorithm designed and made by Saaiq Abdulla Saeed - SAS. This encryption supports any text based files and pure plain text that are in ASCII format.

How to setup SAS-STE

1. Download your desired release of SAS-STE for your operating system under 'Releases' below

2. Extract the downloaded zip file and follow the instructiona in the 'instructions.txt' file
